Первая программа

📖 Теория~10 минут100 XP

Python — это язык программирования. Ты пишешь команды, компьютер их выполняет. Всё просто.

Сначала установи Python: иди на python.org, скачай версию 3.11+. При установке на Windows поставь галочку "Add Python to PATH". Потом установи редактор VS Code — code.visualstudio.com.

Первая команда которую знает каждый программист — print(). Она выводит текст на экран.

main.py
print("Привет, мир!") print("Я учу Python") print(42) print(3.14)
Вывод
Привет, мир! Я учу Python 42 3.14
💡
Запомни: всё что в кавычках "вот так" — это текст (строка). Числа пишутся без кавычек. Скобки () — обязательны!

Функция input() позволяет спросить что-то у пользователя. Он вводит текст с клавиатуры, а Python сохраняет его в переменную.

hello.py
# Знак # — это комментарий. Python его игнорирует. имя = input("Как тебя зовут? ") print("Привет,", имя)
Вывод (имя = "Алишер")
Как тебя зовут? Алишер Привет, Алишер

имя здесь — это переменная. Представь её как коробку с именем: кладёшь внутрь данные, потом используешь по имени. Об этом подробно — в следующем уроке.

Переменные и типы данных

📖 Теория~12 минут100 XP

Переменная — это имя для хранения данных. Создаёшь просто: имя = значение. Знак = — это присваивание, не «равно».

variables.py
имя = "Алишер" # str — строка (текст) возраст = 20 # int — целое число рост = 1.75 # float — дробное число студент = True # bool — да/нет (True/False) print(имя) print(возраст) print(рост) print(студент) print(type(имя)) # покажет тип переменной
Вывод
Алишер 20 1.75 True <class 'str'>

Есть 4 основных типа которые ты будешь использовать постоянно:

str — строка, текст в кавычках: "привет", 'мир'
int — целое число: 42, -7, 1000
float — дробное число: 3.14, -0.5
bool — логическое: только True или False

f-строки — вставляем переменные в текст

Самый удобный способ вставить переменную в строку — f-строка. Пиши f"..." и вставляй переменные в фигурных скобках {}:

fstrings.py
имя = "Алишер" возраст = 20 город = "Ташкент" print(f"Привет, меня зовут {имя}!") print(f"Мне {возраст} лет, я из {город}.") print(f"Через 5 лет мне будет {возраст + 5} лет.")
Вывод
Привет, меня зовут Алишер! Мне 20 лет, я из Ташкент. Через 5 лет мне будет 25 лет.
💡
Важно: внутри {} можно писать не только переменные, но и целые выражения — например {возраст + 5} или {имя.upper()}.

Числа из input() приходят как строки — их нужно преобразовать:

convert.py
возраст_текст = input("Твой возраст: ") # "20" — строка! возраст = int(возраст_текст) # 20 — число следующий = возраст + 1 print(f"В следующем году тебе {следующий}")
Вывод (ввод: 20)
Твой возраст: 20 В следующем году тебе 21

Условия if / elif / else

📖 Теория~12 минут100 XP

Условия позволяют программе принимать решения: если что-то верно — делай одно, иначе — другое.

conditions.py
возраст = 17 if возраст >= 18: print("Ты совершеннолетний") elif возраст >= 14: print("Тебе от 14 до 17") else: print("Тебе меньше 14")
Вывод
Тебе от 14 до 17
⚠️
Отступы — это важно! Python определяет блоки кода по отступам. После if ...: нужен отступ (4 пробела или Tab). Без отступа — ошибка!

Операторы сравнения:

== равно  ·  != не равно  ·  > больше  ·  < меньше  ·  >= больше или равно  ·  <= меньше или равно

and_or.py
пароль = "qwerty123" длина = len(пароль) if длина >= 8 and длина <= 20: print("Длина пароля нормальная") if "123" in пароль or "qwerty" in пароль: print("⚠ Слабый пароль!")
Вывод
Длина пароля нормальная ⚠ Слабый пароль!

Циклы for и while

📖 Теория~12 минут100 XP

Циклы нужны чтобы повторять действия. Без них пришлось бы писать одно и то же сотни раз.

for_loop.py
# for — перебор по диапазону for i in range(5): # i = 0, 1, 2, 3, 4 print(f"Шаг {i}") print("---") # перебор слов в списке пароли = ["admin", "password", "123456"] for пароль in пароли: print(f"Проверяем: {пароль}")
Вывод
Шаг 0 Шаг 1 Шаг 2 Шаг 3 Шаг 4 --- Проверяем: admin Проверяем: password Проверяем: 123456

while — крутится пока условие верно. Опасен бесконечным циклом — всегда думай как он остановится.

while_loop.py
счётчик = 0 while счётчик < 5: print(f"Попытка {счётчик + 1}") счётчик += 1 # то же что счётчик = счётчик + 1 print("Цикл завершён")
Вывод
Попытка 1 Попытка 2 Попытка 3 Попытка 4 Попытка 5 Цикл завершён

break — выход из цикла немедленно. continue — пропустить текущий шаг и перейти к следующему.

break_continue.py
пароли = ["wrongpass", "admin", "qwerty"] верный = "admin" for п in пароли: if п == верный: print(f"Найден! Пароль: {п}") break # нашли — выходим print(f"Не подошёл: {п}")
Вывод
Не подошёл: wrongpass Найден! Пароль: admin

Функции def

📖 Теория~12 минут100 XP

Функция — это именованный блок кода. Написал один раз, вызываешь сколько угодно. Делает код чистым и без повторений.

functions.py
def поздороваться(имя): print(f"Привет, {имя}!") поздороваться("Алишер") поздороваться("Python") поздороваться("Мир")
Вывод
Привет, Алишер! Привет, Python! Привет, Мир!

Списки list

📖 Теория~12 минут100 XP

Список — хранит несколько значений в одной переменной. Незаменим для хранения IP-адресов, паролей, данных.

lists.py
ip_адреса = ["192.168.1.1", "10.0.0.1", "172.16.0.1"] print(ip_адреса[0]) # первый элемент print(ip_адреса[-1]) # последний элемент print(len(ip_адреса)) # количество элементов ip_адреса.append("8.8.8.8") # добавить в конец print(ip_адреса)
Вывод
192.168.1.1 172.16.0.1 3 ['192.168.1.1', '10.0.0.1', '172.16.0.1', '8.8.8.8']

Основные методы списка: append(x) добавить, pop() удалить последний, remove(x) удалить значение, sort() отсортировать, in — проверить наличие.

list_methods.py
числа = [5, 2, 8, 1, 9, 3] числа.sort() print(числа) # [1, 2, 3, 5, 8, 9] print(8 in числа) # True print(7 in числа) # False # срез: [от:до] — получить часть списка print(числа[1:4]) # [2, 3, 5]
Вывод
[1, 2, 3, 5, 8, 9] True False [2, 3, 5]

Словари dict

📖 Теория~12 минут100 XP

Файлы и модули

📖 Теория~12 минут100 XP

Работа с файлами — основа любого инструмента безопасности. Читаем логи, сохраняем результаты, загружаем словари паролей.

files.py
# Запись в файл with open("отчёт.txt", "w", encoding="utf-8") as f: f.write("Результаты сканирования:\n") f.write("Порт 80: открыт\n") f.write("Порт 22: открыт\n") # Чтение из файла with open("отчёт.txt", "r", encoding="utf-8") as f: содержимое = f.read() print(содержимое)
Вывод
Результаты сканирования: Порт 80: открыт Порт 22: открыт
💡
with open() — лучший способ работы с файлами. Файл автоматически закроется когда блок with закончится. Режимы: "r" читать, "w" писать, "a" добавлять.

Вывод
73 b /home/user/projects

Генератор паролей с сохранением
Используй import random и import string. Напиши функцию gen_pass(n) которая создаёт пароль длиной n из букв и цифр. Сгенерируй 3 пароля длиной 12 и выведи их.
task_8.py
import random, string def gen_pass(n): chars = string.ascii_letters + string.digits return ''.join(random.choice(chars) for _ in range(n)) for i in range(3): print(gen_pass(12))
